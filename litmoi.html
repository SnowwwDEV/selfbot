Dernières informations

Prefix du bot par défaut : _

Listes des commandes :

- forcestop / arret / stop (Eteindre le selfbot de force)
- statut / statut-game (Permet de modifier le statut de jeu du bot)
- eval (Permet d'évaluer du code)
- spamfarm (Permet de spamfarm dans le salon où vous avez fait la commande.)
- say / dire (Permet d'envoyer un message en tant que le selfbot)
- ping / pong (Permet de savoir le ping du bot)
- spam (Permet de spam doucement un message)
- hastebin / haste (Permet de crée un lien hastebin avec le texte de votre choix)

Il est déconseillé de toucher au fichier index.js

Heberger le selfbot soi même

ChromeOS

Installez Linux sur votre appareil ChromeOS et ouvrez le terminal Linux, Dedans faites npm i discord.js@11.6.4, npm i hastebin.js, npm i node-fetch et npm i chalk (Faites ça pour la première fois uniquement) et pour lancer le selfbot faites toujours dans le terminal node index.js (Vous devez être dans le dossier du selfbot là ou se trouve index.js, config.json et readme.html) OU suivez les instructions pour Android


Windows

Ouvrez l'invite de commandes (Windows + R et écrivez "cmd"), Dedans faites npm i discord.js@11.6.4, npm i hastebin.js, npm i node-fetch et npm i chalk (Faites ça pour la première fois uniquement) et pour lancer le selfbot faites toujours dans l'invite de commandes node index.js (Vous devez être dans le dossier du selfbot là ou se trouve index.js, config.json et readme.html)


MacOS, Ubuntu, Arch, Manjaro etc

MacOS je sais pas du tout mais Ubuntu, Arch, Manjaro, Kali et la plupart des autres distributions Linux ça doit être en peu comme pour ChromeOS...

Android

Installez l'app Termux depuis le Play Store (Si tu l'as pas bah rip) sur votre appareil et ouvrez cette app (Pensez à bien lui mettre la permission d'accèder aux fichiers), Dedans faites npm i discord.js@11.6.4, npm i hastebin.js, npm i node-fetch et npm i chalk (Faites ça pour la première fois uniquement) et pour lancer le selfbot faites toujours dans cette app node ../../../../../../../../../../../../../../../../../../../sdcard/index.js (Si le fichier index.js et config.json sont au dossier "principal" de votre appareil).
